# Miscellaneous

This folder contains miscellaneous scripts, functions and data used in
scientific papers related to FoodMicrobionet

1.  **make_seq_df** will take as an input a taxonomic reference database
    file in fasta format (click
    [here](https://benjjneb.github.io/dada2/training.html) for examples)
    and return a data frame

2.  **tree_plots_LS2.Rdata** contains phylogenetic ML trees for sequences of
    pathogenic microorganisms, extracted from FoodMicrobionet and
    reference sequences extracted from SILVA v138.1. The trees are
    ggplot2 objects and can be printed with print() or combined with
    cowplot::plot_grid()

3.  **filter_my_taxa_function** is a rather complex function for filtering
    taxa in a phyloseq object using prevalence and/or abundance or a
    permutation filter; it requires PERFect_functions (which, in turn,
    creates the function of the PERFect package, which is not maintained
    anymore and is available
    [here](https://github.com/katiasmirn/PERFect) (but can be replaced
    by the PERFect_functions script), and pc_prevab

4.  **convert_to_microbiome_analyst** folder contains an utility script
    (with three input file provided as an example) which allow to
    convert otu, taxa and sample table extracted from FoodMicrobionet to
    a otu table and a metadata table which can be used with
    [MicrobiomeAnalyst](https://www.microbiomeanalyst.ca/).

    1.  Go to the marker section [data analysis uplod
        window](https://www.microbiomeanalyst.ca/MicrobiomeAnalyst/upload/OtuUploadView.xhtml)

    2.  use the <Choose button> to select the otu table generated by the
        script in the OTU/ASV table section and check "Taxonomy
        included".

    3.  use the <Choose button> to select the sample metadata table
        generatede by the script.

    4.  under Taxonomy labels select "Not specific/Other"
    
5.  **run_ShinyFMBN_from_GitHub** downloads and run ShinyFMBN from GitHub. 
**be patient** it might require package installation in your system  

6.  **network_analysis_v2.1.ipynb** is a Jupyter notebook which allows you to install 
packages needed for microbial association network analysis inference and analysis 
with NetCoMi [![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/ep142/FoodMicrobionet/blob/master/miscellaneous/network_analysis_v2_1.ipynb)
